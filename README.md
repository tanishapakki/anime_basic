**AnimeFunSite – API Integration Assignment**

A mini web application built for the GLOBAL TREND – API Integration Internship demonstrating API consumption, data caching, filtering, error handling, and structured presentation using React and the Jikan Anime API.

**Technologies Used**

Area	Technology
Framework	React.js
Styling	TailwindCSS
Animations	Framer Motion
API	Jikan (MyAnimeList) REST API
Caching	LocalStorage
Build Tool	Vite / Create React App (depending on your setup)

**How to Run the Project**

Prerequisites

Node.js v16+
npm or yarn installed


Installation
git clone <your-repo-link>
cd <project-folder>

npm install
# or
yarn install

Run the development server
npm run dev
# or
yarn dev

Build for production
npm run build

Conclusion

This application fulfills all requirements of the GLOBAL TREND API Integration Internship Assignment, with additional UI/UX enhancements and professional-level API handling.
